# TheChest.Actions

Reusable composite GitHub Actions for .NET and release workflows.

## Available actions

- ✅ [`dotnet-build`](./dotnet-build/README.md): Restore/build .NET projects with optional artifact upload.
- ✅ [`dotnet-test`](./dotnet-test/README.md): Run tests with optional coverage.
- 🧪 [`get-version`](./get-version/README.md): Parse a version from branch/tag refs.
- 🚧 [`publish-nuget`](./publish-nuget/README.md): Pack and publish NuGet packages.
- ✅ [`upload-release-artifacts`](./upload-release-artifacts/README.md): Upload binaries to GitHub Releases.
